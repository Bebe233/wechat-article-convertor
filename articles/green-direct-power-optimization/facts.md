# 事实表 — 绿电直连优化测算系统

> 调研来源：`http://127.0.0.1:5050/` 页面与 `/static/js/app.js`（本地 curl 可读）；公开访问入口 [http://1.13.182.187:5050/](http://1.13.182.187:5050/) 与本地为同一产品界面（外部检索与本地 HTML 标题一致）。**未提供**后端/优化模型源码路径，技术细节以界面与前端 API 为准。

## 产品一句话

面向「风光直连、储能与氢能协同」场景的 Web 测算工作台：按省区与接入电压配置 8760 负荷与设备参数，调用底层 **MILP（混合整数线性规划）** 优化引擎（界面标注 **CPLEX**）完成绿电直连方案测算，并输出度电成本、下网占比、弃电率及全年调度曲线等结果。

英文副标题（页面品牌区）：*Green Direct Power & Hydrogen Synergy Optimization Dashboard*。

## 功能列表

### 计算准备

- **目标省区市**、**接入电压等级** 选择；变更时通过 `/api/province_params` 联动加载默认敏感性电价等参数
- **8760 曲线数据源**：
  - 系统内置典型省区曲线
  - 用户自定义 8760 曲线（Excel/CSV），支持拖拽上传（`/api/upload_curves`）、下载曲线模板
  - **直接导入 `input.xlsx` 工作簿**（`/api/import_input_workbook`）
- **典型负荷类型**（由 `/api/init_data` 下发选项，登录后加载）
- **最大负荷（万千瓦）** 输入
- **允许用户自定义输入参数**（开关）
- **自定义电价与敏感性参数** 配置区
- **设备及技术配置**：数据源默认 `input.xlsx`；支持 **简版 / 全部详细信息** 视图切换；可 **导入 input**（`/api/import_tech_workbook`）；弹窗可 **全部展开/折叠、一键重置默认**，完整展示工作簿中设备、技术、系统控制与分时电价参数

### 优化测算与过程可视

- **启动优化测算**（`/api/simulation/submit`），提交省区、电压、负荷类型、最大负荷、数据源类型等
- 右侧 **CPLEX SOLVER ENGINE ACTIVE CONSOLE**：实时展示 MILP 求解器迭代与分支定界收敛日志（`/api/simulation/task/{id}` 轮询）
- 全局状态：系统就绪 / 计算中等（头部状态指示）

### 结果展示

- 核心 KPI（界面标签）：
  - **度电成本**（前端映射字段含「度电成本（带网）」「度电成本（不带网）」）
  - **下网部分加价**（元/kWh）
  - **电网下网电量占比**（%）
  - **直连弃电率（不含上网）**（%）
- **8760 小时联合调度曲线**（ECharts）：全年 8760h / 春夏秋冬典型周切换
- **优化资产装机容量配置**（测算完成后展示）
- **综合度电成本分项拆分**（饼图，`cost_structure`）
- **综合度电成本测算分项拆分明细表**：费用类别、具体科目、计费基数口径、价格单价/技术参数、年费用、度电分摊、基数度电、政策与计费说明
- **综合电价核算汇总**表
- **导出结果 Excel**（`/api/download`，有结果后启用）

### 任务与协作

- **计算队列**：正在计算、排队任务（`/api/simulation/queue`）
- **历史测算记录**：刷新、条数统计（`/api/user/history`）；独立「历史记录」Tab 在界面中存在但默认 `display:none`
- **用户管理**（需登录）：新增用户（用户名、显示名、初始密码、权限：**操作员 / 查看员 / 管理员**）；用户列表含权限、状态、更新时间；`/api/users` CRUD
- **登录/退出**（`/api/auth/login`、`/api/auth/logout`、`/api/auth/session`）

## 技术亮点

- **8760 小时**全年时序与典型周切片，贴合电力负荷与新能源出力分析习惯
- **Excel/CSV 工作流**：曲线模板、自定义曲线、`input.xlsx` 一体化导入，降低与现有测算表格的衔接成本
- **CPLEX + MILP** 公开在控制台日志中，便于专业人员理解求解过程（非黑盒一键出数）
- **ECharts** 大屏：联合调度曲线、成本结构饼图、资产装机展示
- **多角色权限**与任务队列，适合团队共用同一测算服务（需核实部署方式）

## 差异化（相对「手工 Excel + 离线算例」）

- 省区 × 电压 × 负荷类型 **参数联动**，减少手工抄录默认电价/敏感性
- **同一工作台**完成参数准备、队列提交、日志跟踪、结果图表与 Excel 导出
- **度电成本拆分明细表** 对齐政策与计费口径字段，便于汇报与复核
- 英文副标题强调 **氢能协同**，与纯风光消纳工具形成场景差异（具体氢能变量以 `input.xlsx` 与优化模型为准，见待核实）
